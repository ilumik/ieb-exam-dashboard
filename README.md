# IEB Exam Dashboard

Comprehensive study companion for **Innovation, Entrepreneurship & Business Models** (BSc Economics & Business Administration, 4th semester, Aalborg University).

[

## 🚀 Features

- **25 core exam topics** organized by the 6 course categories
- **69-term glossary** with definitions + real-world use cases  
- **Instant PDF viewer** — opens directly to referenced textbook/lecture pages
- **Deep linking** — click "Oslo Manual p.44" → jumps to exact PDF page 46
- **Smart search** across topics + glossary
- **Fully responsive** — works on desktop, tablet, mobile
- **Offline-first** — all content loads instantly, PDFs from GitHub raw

## 📚 Content Coverage

| Category | Topics | Glossary Terms |
|----------|--------|---------------|
| Innovation | 8 | 16 |
| Entrepreneurship | 7 | 17 |
| Business Models | 5 | 14 |
| Frameworks & Tools | 3 | 12 |
| Critical Theories | 2 | 10 |

**Sources integrated:**
- Trott textbook (corrected page offsets: book p.1 = PDF p.27)
- 4x lecture PDFs (converted from PPTX)
- Oslo Manual 2018, course articles (Wright/Zahra, Haag 2013, etc.)
- 64+ source links with precise page references

## 🎯 Exam Compliance

✅ **Allowed aids** per course description:
> *"Allowed aids: including textbooks, teaching slides, notes from teaching."*

This dashboard = structured **notes from teaching** + direct links to permitted materials.

❌ **Prohibited**: Communication with others, generative AI during exam.

## 📱 Usage

1. **Deploy instantly**: GitHub Pages enabled → https://ilumik.github.io/ieb-exam-dashboard/
2. **Download**: Save `index.html` for offline use
3. **Exam mode**: Open PDFs in new tabs via modal's "↗ Open in new tab" button

## 🛠️ Tech Stack

- **Vanilla HTML/CSS/JS** — no frameworks, no build step
- **PDF.js viewer** (Mozilla) — native browser PDF rendering w/ `#page=X` deep links
- **Responsive CSS Grid** — perfect on all devices
- **Zero dependencies** — pure client-side
